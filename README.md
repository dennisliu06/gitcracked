# GitCracked 

**Interview with an AI Agent** - Your personal coding interview practice platform

Practice LeetCode problems, system design, and OOP skills while talking through your solutions with an AI interviewer. GitCracked simulates real technical interviews to help you build confidence and improve your problem-solving communication.

##  Features

-  **Voice-Powered Interviews** - Practice explaining your thought process out loud, just like in real interviews
-  **Live Code Execution** - Write and test your solutions in real-time with instant feedback
-  **Multiple Interview Types**
  - LeetCode-style coding challenges
  - System design discussions
  - Object-oriented programming concepts
-  **AI Interviewer** - Get intelligent follow-up questions and hints
-  **Real-time Feedback** - Receive instant evaluation on your code and explanations
-  **Modern UI** - Clean, accessible interface built with shadcn components

##  Tech Stack

- **[Next.js](https://nextjs.org/)** - React framework for production
- **[shadcn/ui](https://ui.shadcn.com/)** - Beautiful, accessible component library
- **[Judge0](https://judge0.com/)** - Code execution engine supporting multiple languages
- **[Deepgram](https://deepgram.com/)** - AI-powered voice recognition and synthesis
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
